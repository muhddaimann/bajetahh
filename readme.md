# Expo Web Starter Template

A reusable mobile-first Expo Web template focused on building app-like web experiences using React Native Web.

Built for rapid development of:

- internal systems
- staffing apps
- dashboards
- SaaS platforms
- AI tools
- CRUD applications

---

# Goals

This project exists to explore and standardize:

- Expo Web architecture
- mobile-first responsive layouts
- reusable app structure
- scalable navigation patterns
- reusable providers/contexts
- theme management
- deployment-ready setup

---

# Key Objectives

## 1. Mobile-First Web Experience

Ensure web behaves like a real mobile application:

- centered layout
- constrained max width
- app-like navigation
- touch-friendly UI
- consistent spacing system

---

## 2. Reusable Architecture

Provide reusable patterns for:

- authentication
- overlays/modals
- tab management
- theme system
- token/design system
- API integration
- protected routes

---

## 3. Reusable UI System

Create reusable components for:

- cards
- lists
- forms
- modals
- navigation
- dashboards
- data displays

---

## 4. Production Readiness

Prepare template for:

- Vercel deployment
- Dockerization
- VPS deployment
- self-hosting
- AI integrations

---

# Web Philosophy

This project intentionally treats web as:

```txt
mobile app rendered in browser
```

instead of:

- traditional desktop website
- SEO-first website

Recommended usage:

- internal systems
- dashboards
- HRMS
- staffing systems
- admin apps
- SaaS portals
